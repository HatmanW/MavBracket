About This Project

A fast-start Django base with a tournament module called MavBracket.

Features
  • Single-elimination brackets up to 256 teams
  • Zoomable, touch-friendly SVG bracket
  • Team management (2 players/team) with user linking
  • Admin seeding, advancing, and team swapping
  • Role-aware UI (public view vs. admin controls)
