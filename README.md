# MavBracket
THis is a tournament Bracket to be used for various clubs and fundraisers.
TO DOWNLOAD + RUN:
  1. In directory of your choice, run:
     ```
     git clone https://github.com/HatmanW/MavBracket/edit/main/README.md
     python3 -m venv venv
     ```
     ```
     source venv/bin/activate
     ```
     ``` 
     pip install -r requirements.txt
     ``` 
     ``` 
     pip freeze
     ``` 
     ``` 
     python3 manage.py runserver
     ```
     
   3. On browser of your choosing, navigate to localhost:8000
   4. Take a deep breath
